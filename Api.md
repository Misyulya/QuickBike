# API

### readShops

Возвращает список магазинов.

#### Request

```javascript
{
  filter:{
    order: {
      from: <str(date)>,
      to: <str(date),
    },
    bike: {
      typeIds: <[int]>,
      count: <int>
    }
      
  }
}
```

#### Response

```javascript
{
 ?ok: {
	shops: [<obj: Shop>, ...],
 },
?errors: [...]
}
```
---
